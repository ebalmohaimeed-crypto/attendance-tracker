# نظام متابعة حضور الأعضاء الخارجيين

تطبيق ويب بسيط لمتابعة حضور الأعضاء الخارجيين خلال الفصل الدراسي (18 أسبوع).

## المميزات

- **إضافة الأعضاء** مع كامل التفاصيل: الاسم، الكلية، اسم المقرر، رمزه، رقم الشعبة، أيام المحاضرة، وقت البداية والنهاية
- **تسجيل الحضور الأسبوعي** بضغطة واحدة لكل عضو (حاضر / غائب)
- **تقارير تلقائية** بنسب الحضور لكل عضو مع تنبيهات لمن تجاوز غيابه 25%
- **تصدير Excel (CSV)** يحتوي جميع بيانات الحضور طوال الفصل
- **حفظ البيانات تلقائياً** في المتصفح (localStorage) — لا تحتاج سيرفر
- **واجهة عربية** كاملة مع دعم RTL

## طريقة الاستخدام

1. افتح ملف `index.html` في المتصفح مباشرة — لا يحتاج سيرفر
2. أضف الأعضاء من تبويب **الأعضاء**
3. في بداية كل أسبوع انتقل لتبويب **الحضور الأسبوعي** وسجّل الحضور ثم اضغط حفظ
4. تابع التقارير والتنبيهات من تبويب **التقارير**

## تشغيل محلي بالـ Live Server

```bash
# إذا عندك Python
python -m http.server 8080

# أو Node.js
npx serve .
```

## حفظ البيانات

البيانات تُحفظ في `localStorage` تلقائياً في المتصفح. لا تُفقد البيانات عند إغلاق الصفحة.  
لمشاركة البيانات بين أجهزة متعددة، استخدم زر **تصدير Excel**.
